# 🎬 YouTube Backend Clone (Work in Progress)

🚧 This project is currently under development

---

## 📌 Overview

This is a backend system inspired by YouTube, built to handle core functionalities like video uploads, user authentication, and content management.

The goal of this project is to design a **scalable backend architecture** similar to real-world applications.

---

## 🚀 Features (Planned & Implemented)

### ✅ Implemented

* User Authentication (Signup/Login)
* JWT-based Authorization
* Basic API Structure (MVC Pattern)

### 🔄 In Progress

* Video Upload System
* Like / Dislike Functionality
* Comments System

### ⏳ Planned

* Subscription System
* Video Streaming Optimization
* Search & Recommendation System

---

## 🛠️ Tech Stack

* **Backend:** Java, Spring Boot
* **Database:** MySQL
* **Authentication:** JWT
* **Tools:** Postman, Git
